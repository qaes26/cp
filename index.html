Figure 1: Recursion and Summation

Recursion Functions

Visual Content: The initial pages show a C++ code snippet defining a recursive function named sum.

int sum(int x) { if (x == 1) return 1; // Implied base case return x + sum(x - 1); }

Detailed Explanation: The image illustrates recursion. The function sum(int x) calculates the summation from 1 to x. It stops when x reaches 1 (base case) and adds the current number to the result of the function called with x-1.

Figure 2: Prime Number Logic

Loops Algorithms

Visual Content: A function bool isPrime(int x).

bool isPrime(int x) { int counter = 0; for (int i = 1; i <= x; i++) { if (x % i == 0) counter++; } return (counter == 2); }

Detailed Explanation: A brute-force algorithm to check for Prime numbers. It counts how many numbers divide x cleanly. If the count is exactly 2 (1 and itself), it returns true.

Figure 3: Triangle Inequality

Boolean Logic

Visual Content: bool isTriangle(int x, int y, int z).

if (x+y > z && x+z > y && y+z > x) return true;

Detailed Explanation: Checks if three lengths can form a triangle. All three conditions (sum of any two sides > third side) must be true using the AND (&&) operator.
